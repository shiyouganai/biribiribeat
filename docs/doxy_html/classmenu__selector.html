uint8_t menu_selector::select ( void  )

Select the current menu entry.

The menu selector will, when select is called, return the numerical value of the entry that the cursor is currently on. For example, if the cursor is on entry 3, and select is called, 3 will be returned.

Returns
The entry the cursor is currently selecting.
